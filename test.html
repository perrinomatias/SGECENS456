Este es un sistema completo de gestión para el CENS 456 Ezeiza que incluye:

  • Sistema de autenticación seguro
  • Dashboard de gestión con interfaz intuitiva

  • Diseño responsive para móviles
  • Control de acceso por roles (4 niveles)
  • Sistema específico de carga de notas
  • Integración con WhatsApp para contacto directo
  • ¡NUEVO! Sistema de registro de usuarios
  • ¡NUEVO! Gestión completa de estudiantes
  • ¡NUEVO! Gestión completa de docentes
  • ¡NUEVO! Sistema de reportes con gráficos
  • ¡NUEVO! Panel de configuración de perfil

Sitio Web Principal Sistema de Login Dashboard (requiere login) Registro de Usuarios

Google Apps Script Configurados

El sistema está configurado para acceder a estos 3 Google Apps Script:

1. Sistema Principal: /exec
2. Carga de Notas: /exec?page=carga
3. Sistema de Alertas: /exec?page=alertas

Nota: Los Apps Script se abren automáticamente desde el dashboard una vez que te loguees.

Sistema de Control de Acceso

El sistema implementa control de acceso basado en roles (RBAC):

  • Administrador: Ve todas las opciones (Gestión + Notas + Alertas)
  • Preceptor/DICYT: Solo ve Gestión y Alertas
  • Profesor: Solo ve Carga de Notas
Tip: Prueba iniciar sesión con diferentes roles para ver cómo cambia la interfaz automáticamente.

Cómo Probar el Sistema

  1. Visita el sitio web principal para ver la página institucional
  2. Haz clic en "Sistema de Gestión" en el menú o ve directamente al login
  3. Usa cualquiera de las credenciales de prueba mostradas arriba
  4. ¡Nuevo! Prueba diferentes roles para ver el control de acceso
  5. Explora el dashboard y prueba los enlaces a los Apps Script
  6. Prueba el logout y la funcionalidad de "recordarme"

Dashboard Completamente Funcional

¡Todas las secciones ahora funcionan! El dashboard incluye gestión completa:

  • Gestión de Estudiantes: Agregar, editar, eliminar y buscar estudiantes
